mysql常用命令

发表于 2018-11-13 | 分类于 mysql
  1. 进入控制台并启动mysql服务器(Windows系统)
    以管理员身份运行cmd
    切换到mysql的安装目录 cd C:\Program Files\MySQL\MySQL Server 8.0\bin
    进入mysql操作界面 mysql -u root -p
  2. 查看mysql的版本
    select version();
  3. 查看服务器中存在什么数据库
    show databases;
  4. 创建一个数据库mydb;
    create database mydb;
  5. 选择你所创建的数据库
    use mydb;
  6. 查看数据库中的表
    show tables;
  7. 创建一个数据库表 mytable
    create table mytable(name varchar(20),age varchar(20),sex char(1));
  8. 描述表结构 mytable
    describe mytable
  9. 向 mytable 中插入数据
    insert into mytable(name,age,sex) values ("fjl","26","m");
  10. 查看 mytable 中的数据
    select * from mytable;
  11. 修改 mytable 中的数据
    update mytable set name = "jalenFang" where name = "fjl";
  12. 删除 mytable 中的数据
    delete from mytable where age = "29";
  13. mytable 增加一列 爱好(hobby)
    alter table mytable add column hobby varchar(20);
  14. 删除表 mytable
    drop table mytable
  15. 删除数据库 mydb;
    drop database mydb
  16. mysql 退出行编辑
    '/c
  17. 退出 mysql
    quit

mysql安装以及启动

发表于 2018-11-12 | 分类于 mysql

mysql安装

下载地址
windows上安装参考链接
Mac上双击dmg文件，根据提示安装即可

Mac上 mysql环境变量配置

  1. 打开终端,输入： cd ~
  2. 然后输入：sudo vim .bash_profile，回车执行，需要输入root用户密码。sudo是使用root用户修改环境变量文件
  3. 输入 E 进入编辑模式， 输入 I 就可以输入文字了
  4. 在文档的最下方输入：export PATH=${PATH}:/usr/local/mysql/bin
  5. 然后esc退出insert状态，输入:wq保存退出。
  6. 输入：source .bash_profile，回车执行，运行环境变量。

mysql启动方式

一、Windows上启动方式

  1. 第一种方式
    控制面板 - 管理工具 - 服务
  2. 第二种方式
    cmd(以管理员方式运行) - cd 安装路径(切换到mysql安装的bin目录) - net start mysql80(mysql的名字)

二、Mac上启动方式
系统偏好设置 - 点击mysql

错误解决

  • 启动mysql，发生系统错误5。拒绝访问。
    打开cmd，需要以管理员身份运行

hexo常用命令

hexo常用命令

  • 安装hexo
    npm install hexo-cli -g

  • 安装hexo部署到GitHub Pages的Deployer
    npm install hexo-deployer-git --save

  • 创建hexo文件夹
    hexo init

  • 在本地查看效果
    hexo g
    hexo s

  • 效果满意，部署到GithHub
    hexo g -d

  • 创建文章
    hexo n "文章的名字"
    或者直接在source/_posts文件下新建文章

  • 新建标签页
    hexo new page "tags"

  • 新建分类页
    hexo new page "categories"
